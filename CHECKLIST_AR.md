# ✅ قائمة التفقد الشاملة - خطوة بخطوة

## المرحلة 1️⃣: التحضير

### قبل البدء:
- [ ] تأكد إنك متصل بـ نت مستقر
- [ ] جهز كل الأجهزة اللي بتستخدم التطبيق (الهاتف، التابلت، الحاسوب)
- [ ] احفظ أهم بياناتك (اطلب تقرير PDF من الموقع)
- [ ] فتح حسابك على Netlify في متصفح

---

## المرحلة 2️⃣: مسح الـ Cache من الأجهزة

### على الهاتف الذكي (Android):
- [ ] افتح Chrome وروح لموقع التطبيق
- [ ] اضغط ثلاث نقاط في الأعلى → Settings
- [ ] اختر Site settings → Cookies and site data
- [ ] تفحص الموقع واختر "Clear and reset"
- [ ] أغلق الـ Chrome تماماً
- [ ] أعد فتحه والدخول للموقع

### على الهاتف الذكي (iPhone):
- [ ] افتح Safari
- [ ] اذهب للموقع
- [ ] اضغط على address bar
- [ ] ستظهر خيارات → اضغط "Clear History and Website Data"
- [ ] اختر "All time" و "All websites"
- [ ] اضغط "Clear"

### على الكمبيوتر (Windows/Mac):
- [ ] افتح Chrome
- [ ] اضغط F12 (Developer Tools)
- [ ] اذهب للـ "Application" tab
- [ ] في اليسار، اضغط "Service Workers"
- [ ] اختر الموقع واضغط "Unregister"
- [ ] في اليسار، اضغط "Storage"
- [ ] على اليمين بتحت "Clear site data"
- [ ] تأكد إن جميع الخيارات محددة
- [ ] اضغط "Clear All"
- [ ] أغلق F12
- [ ] اضغط Ctrl+F5 (أو Cmd+Shift+R على Mac)

## المرحلة 3️⃣: رفع الملفات الجديدة

### الطريقة 1: Drag & Drop (الأسهل):
- [ ] اذهب لـ https://app.netlify.com
- [ ] ادخل على حسابك
- [ ] اختر المشروع "Central Kitchen 1980"
- [ ] ابحث عن "Deploys" و انقر على "Deploy new site"
- [ ] اسحب هذه الملفات الـ 5 على المربع:
  - [ ] index.html
  - [ ] netlify.toml
  - [ ] manifest.json
  - [ ] service-worker.js
  - [ ] qr.html
- [ ] انتظر حتى تظهر رسالة "Published"

### الطريقة 2: Git Push (لو معك git):
- [ ] افتح terminal/command prompt
- [ ] ادخل مجلد المشروع
- [ ] اكتب: `git add .`
- [ ] اكتب: `git commit -m "fix: resolve cache and data persistence issues"`
- [ ] اكتب: `git push origin main`
- [ ] اذهب ل Netlify وتفرج على الـ deploy

## المرحلة 4️⃣: التحقق من النشر

### على Netlify Dashboard:
- [ ] اذهب لـ "Deploys"
- [ ] يجب أن ترى "Published" في الأخضر
- [ ] لو ما قال "Published"، انتظر دقيقة وأعد تحميل الصفحة
- [ ] انسخ الرابط الجديد (آخر deploy)

### في موقع التطبيق:
- [ ] افتح الرابط في متصفح جديد (ليس tab قديم)
- [ ] انتظر 3-5 ثواني للتحميل الكامل
- [ ] يجب أن ترى واجهة التطبيق بدون أخطاء

---

## المرحلة 5️⃣: اختبار الإصلاحات

### اختبار 1: هل الموقع يفتح؟
- [ ] جرب فتح الموقع من متصفح نظيف (بدون cache)
- [ ] يجب أن يفتح في أقل من 5 ثواني
- [ ] اذا فيه تأخير، اضغط Ctrl+F5 للتحديث القوي

### اختبار 2: هل البيانات محفوظة؟
```javascript
// افتح console (F12) واكتب:
console.log('Items:', items.length);
console.log('History:', orderHistory.length);
```
- [ ] يجب أن ترى رقم الأصناف والطلبيات
- [ ] لو قال "undefined"، أعد تحميل الصفحة

### اختبار 3: هل تقدر تضيف صنف جديد؟
- [ ] اذهب لصفحة الكتالوج
- [ ] اضغط على إضافة صنف جديد
- [ ] ادخل اسم اختياري (مثل "تجربة")
- [ ] اضغط حفظ
- [ ] يجب أن تظهر رسالة خضراء "✅ تمت الإضافة"
- [ ] أعد تحميل الصفحة (F5)
- [ ] تأكد إن الصنف الجديد موجود بعد التحميل

### اختبار 4: هل تقدر تحذف صنف؟
- [ ] اختر أي صنف وحذفه
- [ ] تأكد من الحذف
- [ ] يجب أن تظهر رسالة خضراء "🗑️ تم حذف"
- [ ] أعد تحميل الصفحة
- [ ] تأكد إن الصنف محذوف فعلاً

### اختبار 5: هل الطلبيات محفوظة؟
- [ ] اضغط على عمل طلبية جديدة
- [ ] أضف كمية من أي صنف
- [ ] اضغط حفظ
- [ ] اذهب لصفحة السجل
- [ ] يجب أن تشوف الطلبية الجديدة
- [ ] أعد تحميل الصفحة
- [ ] تأكد إن الطلبية بعد موجودة

## المرحلة 7️⃣: حل مشاكل إضافية

### لو الموقع ما بينفتح:
- [ ] جرب متصفح تاني (Safari, Edge, Firefox)
- [ ] تأكد من الاتصال بالنت
- [ ] اضغط Ctrl+Shift+Delete (مسح cache متقدم)
- [ ] افتح console (F12) وشوف الأخطاء الحمراء

### لو البيانات ما توجد:
```javascript
// في console:
localStorage.clear();
location.reload();
```
- [ ] جرب هذا وانتظر تحميل جديد

### لو Firebase ما يتصل:
```javascript
// في console:
console.log('Firebase:', window._db ? '✅' : '❌');
```
- [ ] لو قال ❌، أعد تحميل الصفحة
- [ ] أنتظر 30 ثانية للاتصال التلقائي

## 📞 حل المشاكل الطوارئ:

### الموقع بطيء جداً:
```
1. أغلق جميع التبويبات الأخرى
2. امسح الـ Browser Cache
3. جرب متصفح تاني
4. تأكد من سرعة النت
```

### البيانات لا تحفظ:
```
1. تحقق من Firebase Connection
2. جرب إضافة عنصر جديد وانتظر الحفظ
3. أعد التحميل وتأكد إن العنصر موجود
```

### العناصر تختفي:
```
1. افتح Console (F12)
2. اكتب: console.log(deletedItemNames);
3. تحقق من قائمة المحذوفات
4. إذا كان اسم العنصر في القائمة، أزله يدويًا
```
